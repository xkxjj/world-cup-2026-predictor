# 2026 世界杯预测器

这是一个基于 Elo 评分 + 蒙特卡洛模拟的 2026 世界杯预测器。

在线访问：

https://xkxjj.github.io/world-cup-2026-predictor/

## 功能

- 夺冠概率预测
- 小组出线概率
- 对阵比分预测
- 上下半场比分路径预测
- 黄牌、红牌、越位、角球、犯规等事件预测
- 公允投注倍率参考
- 已赛结果验真
- 自动同步线上赛果

## 数据与模型

比分预测基于：

- 国家队 Elo 评分
- 2026 世界杯分组与赛程
- 已赛真实比分
- 泊松分布比分概率
- 蒙特卡洛完整赛程模拟

页面会在打开时自动同步线上已赛结果，并每 5 分钟刷新一次。

## 注意

投注倍率只是模型公允赔率参考，不是投注建议。
